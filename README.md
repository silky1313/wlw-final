## 命名规则
```
p 阅览室内的当前人数
t 温度
l 光线亮度
m 进入阅览室总人数
cf 风扇状态 00 - 11
cl 小灯状态 00 -11

dl dr 开放时间
tl tr 温度范围
ll lr 亮度范围
cf 控制风扇
cl 控制灯光
```
